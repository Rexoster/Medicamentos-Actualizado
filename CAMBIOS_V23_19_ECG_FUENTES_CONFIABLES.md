# v23.19 · ECG con fuentes clínicas confiables

## Cambios

- Se actualizó la bibliografía del módulo ECG.
- Se retiró Wikipedia como referencia documental.
- Se agregó como fuente docente la playlist **Electrocardiografía Clínica 2024** de la Facultad de Medicina UNAM, indicada por el usuario.
- Se reforzaron las fuentes principales con:
  - AHA/ACCF/HRS para estandarización e interpretación ECG.
  - ACC/AHA/HRS 2018 para bradicardia y trastornos de conducción.
  - Fourth Universal Definition of Myocardial Infarction 2018 para infarto/isquemia.
  - NCBI Bookshelf / StatPearls como apoyo secundario.
  - KDIGO para diskalemias.
  - ESC para pericarditis.
- Se corrigió una duplicación accidental de `twelveLeadLayout` detectada al preparar el parche.
