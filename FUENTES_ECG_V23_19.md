# Fuentes ECG v23.19 · sustitución de Wikipedia por fuentes clínicas

## Cambio principal

Se retiró Wikipedia como fuente de respaldo en la documentación del módulo ECG. Wikipedia puede servir como orientación rápida para un humano con insomnio y demasiadas pestañas abiertas, pero no como bibliografía clínica principal dentro de una aplicación médica.

## Fuente docente indicada por el usuario

- Facultad de Medicina UNAM. Playlist de YouTube: **Electrocardiografía Clínica 2024**. Se usa como apoyo docente para organización didáctica, nombres de patrones y forma de explicar criterios.

## Fuentes clínicas principales usadas para criterios ECG

1. **AHA/ACCF/HRS Recommendations for the Standardization and Interpretation of the Electrocardiogram**.
   - Part III: intraventricular conduction disturbances.
   - Part IV: ST segment, T waves and QT interval.
   - Part V: chamber hypertrophy.
   - Part VI: acute ischemia/infarction.

2. **ACC/AHA/HRS 2018 Guideline on Bradycardia and Cardiac Conduction Delay**.
   - Base para BAV 1º, Mobitz I, Mobitz II, BAV 2:1, BAV avanzado y BAV completo.

3. **Fourth Universal Definition of Myocardial Infarction (2018)**.
   - Base para criterios de lesión/isquemia, elevación del ST, derivaciones contiguas y correlación con biomarcadores.

4. **NCBI Bookshelf / StatPearls**.
   - Apoyo secundario para descripciones clínicas de bloqueos AV, bloqueos de rama, WPW, taquiarritmias y alteraciones por potasio.

5. **KDIGO Dyskalemia Consensus / Kidney International**.
   - Apoyo para alteraciones por potasio, recordando que el ECG no estima potasio sérico ni sustituye laboratorio.

6. **ESC Pericardial Disease Guidance**.
   - Apoyo para pericarditis: elevación difusa/cóncava del ST, depresión del PR y cambios en aVR.

## Uso dentro de la app

Las fuentes se usan para:

- describir criterios completos por patología;
- decidir qué criterios se pueden sombrear sobre el ECG didáctico;
- separar criterios visibles de criterios clínicos que no deben pintarse como si fueran foquitos navideños;
- reforzar que el módulo es educativo y no diagnóstico automático.

## Nota de seguridad clínica

El ECG generado es una simulación didáctica. No reemplaza medición manual del ECG real, revisión de 12 derivaciones, tira larga, clínica, biomarcadores, electrolitos séricos ni valoración médica.
